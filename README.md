# EditRush

A browser-based puzzle game that demonstrates the pain of manual video editing vs. AI-powered clipping.

**[Play it here](https://franz23.github.io/EditRush/)**

## The Game

You're a video editor drowning in raw footage. Awkward, oversized content blocks pile up while chaos events hit: revision requests, Premiere crashes, wrong aspect ratios, competitors posting first.

Then you discover OpusClip. Pieces get smaller, smarter, and the board clears itself.

## How to Play

- **Arrow keys** to move and rotate
- **Space** to hard drop
- Survive the chaos, then click **Get OpusClip** when it appears
- Clear the board to win
